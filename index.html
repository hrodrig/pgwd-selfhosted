Helm repository · pgwd

This site hosts the Helm chart index for deploying pgwd (Postgres Watch Dog) on Kubernetes. It is published from pgwd-selfhosted (deployment manifests and chart sources). Helm clients fetch index.yaml from this base URL — a browser is optional.

Add the repo and install

helm repo add pgwd https://hrodrig.github.io/pgwd-selfhosted
helm repo update
helm install pgwd pgwd/pgwd -n pgwd --create-namespace -f my-values.yaml

Create my-values.yaml first (e.g. helm show values pgwd/pgwd > my-values.yaml); use helm upgrade --install for repeat installs.
