# Standalone on macOS

← [Back to run/README](../../README.md).

1. Download the **macOS** tarball for your architecture from [pgwd Releases](https://github.com/hrodrig/pgwd/releases).
2. Extract and run the binary from a terminal; set **`PGWD_DB_URL`** and other **`PGWD_*`** variables (see **[upstream README](https://github.com/hrodrig/pgwd/blob/main/README.md)**).
