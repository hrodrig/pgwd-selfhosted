# Standalone on Windows

← [Back to run/README](../../README.md).

1. Download the **Windows** zip for your architecture from [pgwd Releases](https://github.com/hrodrig/pgwd/releases).
2. Extract **`pgwd.exe`** and run it from **PowerShell** or **cmd**, setting **`PGWD_DB_URL`** and other variables (see **[upstream README](https://github.com/hrodrig/pgwd/blob/main/README.md)**).
